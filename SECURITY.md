# Security Policy

This repository contains curated Rive reference documentation, recipes,
eval cases, and a small set of Python validation scripts (`scripts/`,
`tests/`) plus a few GitHub Actions workflows under `.github/workflows/`.
It has no runtime dependencies, no service code, and no credentials.

If you find a security issue — for example, a workflow that could be
exploited, a script that mishandles untrusted input, or a committed
secret — please open a regular GitHub issue. There is no need for a
private disclosure process.
